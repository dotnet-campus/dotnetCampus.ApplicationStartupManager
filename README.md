# dotnetCampus.ApplicationStartupManager

应用软件的启动流程组件。 支持高性能异步多线程，自动构建启动流程图，动态分配启动任务资源，支持接入预编译框架，支持所有的 .NET 应用。

Application startup manager for all .NET application.
